🔍 Vérification du Format de Réponse Login

📊 Formats de Réponse

❌ Ancien Format (Ne Fonctionne Pas)

{
  "success": true,
  "data": {
    "token": "...",
    "user": {...}
  }
}

Le frontend ne trouve pas response.token

✅ Nouveau Format (Devrait Fonctionner)

{
  "success": true,
  "token": "...",
  "user": {...}
}

Le frontend trouve response.token directement
